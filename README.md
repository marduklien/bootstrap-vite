# Bootstrap Vite 練習專案

Bootstrap 5.3 + Sass + Vite + EJS 多頁面練習專案。

## 技術棧

- Bootstrap 5.3.3
- Sass 1.77.6
- Vite 5.x
- EJS（透過 vite-plugin-virtual-mpa）

## 頁面

- 首頁 (index)
- 關於我們 (about)
- 產品介紹 (product)
- 聯絡我們 (contact)

## 安裝與執行

```bash
npm install
npm run dev
```

## 部署

```bash
npm run deploy
```
